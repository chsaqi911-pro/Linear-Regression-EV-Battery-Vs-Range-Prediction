# Linear-Regression-EV-Battery-Vs-Range-Prediction
In order to predict range (in Km) of an electric vehicle (EV) bike or shootee based on the size of its battery (in AH) this model is trained.
## Libraries

- NumPy
- Pandas
- Scikit-Learn
- Joblib
  ## Model

Linear Regression

## Accuracy

R² Score = 0.77
